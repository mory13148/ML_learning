Neural Network · 60 pts

🌸 Operation: Iris

60 Points
📖 Mission Briefing

Deep in the mountains of a remote research station, a botanist named Dr. Elena Voss has discovered three rare species of flowers that look almost identical to the naked eye. Misidentifying even one specimen could lead to an ecological disaster — one species is a keystone pollinator, another is mildly toxic, and the third is endangered and protected by law.

Dr. Voss has spent years collecting measurements — sepal length, sepal width, petal length, and petal width — for hundreds of flowers, carefully labelling each one. Now she needs a machine to do the classification for her, because the next expedition leaves in hours and she won't be there to watch.

Your job: build a neural network — entirely from scratch — that Dr. Voss can trust with her flowers. No black boxes. She wants to see every calculation laid bare.

You will use the classic Iris dataset (150 samples, 4 features, 3 classes). A ready-to-load version is provided below. Your network must reach ≥ 90% accuracy on a held-out test set.

📂 data_loader.py — provided starter
import numpy as np
from sklearn.datasets import load_iris   # only allowed import for DATA loading

def get_data(test_ratio=0.2, seed=42):
    iris = load_iris()
    X, y = iris.data, iris.target
    # shuffle
    rng = np.random.default_rng(seed)
    idx = rng.permutation(len(X))
    X, y = X[idx], y[idx]
    # split
    n = int(len(X) * ((1 - test_ratio)))
    return X[:n], X[n:], y[:n], y[n:]

Your Tasks

  • 1
    Activation Functions
    Implement ReLU and Softmax from scratch as classes with forward() and backward() methods. Remember: Softmax must be numerically stable (subtract max before exp).
  • 2
    Aggregation (Dense Layer)
    Implement a fully-connected layer that computes z = X · W + b. Store weights W and bias b as instance variables. Initialize W with Xavier / Glorot initialisation.
  • 3
    Loss Function
    Implement Cross-Entropy Loss for multi-class classification. Clip predicted probabilities to avoid log(0).
  • 4
    Backpropagation
    Derive and implement the gradient flow through each layer: Loss → Softmax → Dense₂ → ReLU → Dense₁. Each layer must return dW, db, and dX.
  • 5
    Gradient Descent Optimiser
    Implement vanilla SGD (with an optional momentum extension for bonus credit). Update every weight and bias after each mini-batch.
  • 6
    Training Loop & Evaluation
    Wire everything together. Train for enough epochs to converge. Print training loss every 50 epochs. At the end, print test accuracy. You must reach ≥ 90 %.
💡 Suggested Architecture

Input (4) → Dense (64 neurons) → ReLU → Dense (32 neurons) → ReLU → Dense (3 neurons) → Softmax. Feel free to tweak layer sizes — but justify your choice in a comment.

📐 Key Formulas to Recall
ReLU(z) = max(0, z)
Softmax(zᵢ) = exp(zᵢ − max(z)) / Σⱼ exp(zⱼ − max(z))
L = − (1/N) Σᵢ Σₖ yᵢₖ · log(ŷᵢₖ + ε)
W ← W − η · dW
